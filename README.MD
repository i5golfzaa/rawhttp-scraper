# RawHTTP Scraper

A simple Web scraper based on [RawHTTP](https://github.com/renatoathaydes/rawhttp).

## Usage

```
java -jar rawhttp-scraper.jar <request-file>
```

For example request files, see the [samples/requests](samples/requests) directory.

## Building

```
./gradlew jar
```

## Testing

```
./gradlew check
```
